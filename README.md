Carhop
======

[![Build Status](https://travis-ci.org/lpil/carhop.svg?branch=master)](https://travis-ci.org/lpil/carhop)

Store your samples on Amazon's S3 cloud object storage.


## Development

```sh
# Install the deps
bundle install

# Run the tests
bundle exec rspec

# Run the test watcher
bundle exec guard

# Run the linter
bundle exec guard
```
